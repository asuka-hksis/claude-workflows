# 案件探しワークフロー

## トリガー
ユーザーが「案件を探してください。」と伝えたとき。

## 全体の流れ
SESマッチングシートの案件タイトルが未記入の行を上から5行ずつ処理し、シートの一番下まで繰り返す。

## ステップ

### 1. 対象行の取得
- Notion内の「SESマッチングシート」を開く
- 案件タイトルが空欄の行を上から順に5行分確認する
- 各行の要員情報を取得する

### 2. 案件の検索
取得した要員ごとに、最も見合った案件を以下の2つのソースから探す。

#### ソース1: Gmail（過去2週間分のメール）
- Notion内の「パートナー企業一覧」ページに記載されているメールアドレスから、asuka-higuchi@ex.finn.co.jp 宛に送られたメールを確認
- 過去2週間以内のメールが対象

#### ソース2: Notion（案件シート）
- Notion内の「パートナー企業一覧」ページ内にある案件シートを確認

### 3. SESマッチングシートへの記入
案件が見つかったら、SESマッチングシートの該当行に以下を記入する。

| 項目 | 内容 |
|---|---|
| 案件タイトル | 見つかった案件のタイトル |
| 案件ソース媒体 | どこから見つけたか（Gmail / Notion案件シート） |
| 案件送信元メールアドレス | メール由来の場合、送信元メールアドレスを記入 |
| 案件メール受信日付 | メール由来の場合、メールの受信日時を記入（分まで記入） |
| 案件Sheet該当場所 | 案件シート由来の場合、該当箇所（シート名・位置など）を記入 |

※ 案件送信元メールアドレス、案件メール受信日付、案件Sheet該当場所は必要に応じて記入。

### 4. 要員ページへの案件情報貼り付け
- 各行の「要員名」をクリックして開く
- 既に入力されている要員情報は消さず、その下に続けて参照元のメール本文コピーまたは案件シートの該当部分のコピーを貼り付ける

### 5. 5件分の完了報告
- 5行分の処理が終わったら、SESマッチングシートの一番下にある「時間記録」欄に記録を書く
- 書き方: `○月○日○時○分 No.○〜No.○ 案件出し`
- すでに書いてある内容がある場合は消さず、その下に続けて書く
- 確認を求めず、そのまま次の5件の処理に進む

### 6. 繰り返し
- ステップ1に戻り、次の5行を処理する
- SESマッチングシートの一番下まで繰り返す

## 終了条件
- SESマッチングシートの案件タイトルが空欄の行がすべて処理されたら終了
- 最後の記録も「時間記録」欄に書いて完了
